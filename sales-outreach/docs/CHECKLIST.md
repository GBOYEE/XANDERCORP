> **Created: 2026-04-17**

## CHECKLIST TEMPLATE

Copy this for each day's checklist:

```
═══════════════════════════════════════════
📋 DAILY CHECKLIST — {DATE}
═══════════════════════════════════════════

OUTREACH
[ ] Check email responses (morning + evening)
[ ] Send outreach (max 15/day)
[ ] Follow up on non-responders (day 3, 6, 10)
[ ] Add new leads to database
[ ] Track sent emails in log

SALES
[ ] Respond to any new inquiries
[ ] Send proposals if needed
[ ] Update lead status
[ ] Log any deals closed

CONTENT
[ ] Portfolio updated?
[ ] GitHub activity?
[ ] Social media posts?

SYSTEM
[ ] Server health check
[ ] Backup run?
[ ] Logs reviewed?

REVIEW (End of Day)
[ ] What worked today?
[ ] What didn't work?
[ ] Tomorrow's priorities?

═══════════════════════════════════════════
```

---

## DAILY REVIEW QUESTIONS

Answer these every night before 00:00:

1. **What did I do today?**
2. **What worked?** (Specific)
3. **What didn't work?** (Specific)
4. **What should I do differently tomorrow?**
5. **Any blockers?** (Need CEO input?)
