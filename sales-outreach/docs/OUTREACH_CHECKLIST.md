# XanderCorp Outreach Pre-Send Checklist
> MANDATORY — No exceptions

## ⚠️ CRITICAL RULE

**CEO MUST TEST EVERY EMAIL BEFORE FIRST BATCH**

---

## Pre-Send Checklist (Every Email)

### 1. Content Check
- [ ] Under 60 words?
- [ ] Subject is 2-4 words, personalized?
- [ ] No numbers in subject line?
- [ ] Lead with VALUE, not pitch?
- [ ] Clear CTA (one line)?
- [ ] No attachments?
- [ ] No generic phrases?

### 2. Personalization Check
- [ ] Lead's name correct?
- [ ] Company name correct?
- [ ] Trigger event verified (not stale)?
- [ ] Pain point specific to THEM?
- [ ] No placeholder text remaining?

### 3. Technical Check
- [ ] From email: professional sender?
- [ ] Reply-to: set correctly?
- [ ] Signature: clean, no clutter?
- [ ] Links: verified working?

### 4. Lead Quality Check
- [ ] Company verified real?
- [ ] Lead score 70+ (HOT)?
- [ ] Not on do-not-contact list?
- [ ] Not previously contacted (check log)?

### 5. Test Email (MANDATORY)
- [ ] Sent to self first?
- [ ] Looks good in inbox?
- [ ] Subject line shows correctly?
- [ ] Mobile view tested?
- [ ] CEO approved?

---

## Batch Send Rules

| Condition | Action |
|-----------|--------|
| First ever batch | CEO tests + approves EACH email |
| New template | Test 3x before bulk |
| New lead source | Test 5x first |
| Any doubt | DO NOT SEND — ask CEO |

---

## Daily Limits (Gmail Safe)

| Day | Max Emails |
|-----|-----------|
| Week 1 | 5/day |
| Week 2 | 10/day |
| Week 3+ | 15/day |

**DO NOT EXCEED** — account safety first

---

## If Something Goes Wrong

1. **High bounce rate** → Pause immediately, verify list
2. **Spam complaints** → Pause immediately, review content
3. **Low open rate** → Test new subject lines
4. **No replies** → Review personalization + offer
