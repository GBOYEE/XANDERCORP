# Activity Log

> **Every significant action is logged. Data is truth.**

---

## Log Structure

| Log Type | File | Purpose |
|----------|------|---------|
| Activity | activity.log | All significant actions |
| Revenue | revenue.log | Money in/out |
| Outreach | outreach.log | Sales outreach activity |
| System | system.log | Technical events |
| Decisions | decisions.log | CEO decisions |
| Errors | errors.log | Problems and fixes |

---

## Log Format

```
[TIMESTAMP] | TYPE | ACTOR | ACTION | DETAILS
```

Example:
```
2026-04-17 09:50 | REVENUE | nanobot | Upwork approved | Profile: https://upwork.com/freelancers/~01ed00297dd102af14
```
